# TS ：核心任务清单

这 5 个点涵盖了 TS 基础的全部精华，学完它们，就真正迈过了“静态类型”的门槛。

🚀 TS 阶段一：全量知识点清单 (5小时攻坚版)
1. 基础标量类型 (The Primitives)
(1)知识点：string, number, boolean, null, undefined。
(2)核心实验：在 tsconfig.json 中开启 strictNullChecks，观察为什么不能把 null 赋值给 number。
(3)意义：消除 JS 中最常见的 undefined is not a function 错误。

2. 顶级类型与特殊类型 (Top & Special Types)
(1)知识点：any vs unknown vs void vs never。
(2)核心实验：尝试对 unknown 类型的变量直接调用 .length（报错）。
写一个一定会抛出异常的函数，看它的返回类型是否自动推断为 never。
(3)意义：掌握 TS 的“安全边界”。

3. 集合类型 (Collections)
(1)知识点：数组 T[] 与 元组 Tuple。
(2)核心实验：定义一个只读元组 readonly [number, string]。
尝试用 .push() 破坏元组结构，观察 TS 的拦截机制。
(3)意义：精准描述结构化数据（如 React Hooks 返回值或 API 响应）。

4. 枚举增强 (Advanced Enums)
(1)知识点：数字枚举、字符串枚举、常量枚举 (const enum)。
(2)核心实验：对比普通 enum 和 const enum 编译后的 JS 差异，理解零开销抽象。
(3)意义：提高代码可读性，同时不损失运行性能。

5. 类型断言与字面量类型 (Assertions & Literals)
(1)知识点：as 断言、字面量类型（如 type Role = 'admin' | 'user'）。
(2)核心实验：用 as const 将一个对象变为完全只读。
尝试给字面量联合类型赋值范围之外的字符串。
(3)意义：让类型约束从“大类”（string）精确到“具体值”（'admin'）。
